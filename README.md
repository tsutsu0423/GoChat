# GoChat

Go製のリアルタイムチャットアプリケーション。

## 機能

- ユーザー登録・ログイン・ログアウト
- WebSocket によるリアルタイムメッセージ送受信
- チャット履歴の保存・表示
- 自分のメッセージの削除
- タイピングインジケーター
- オンラインユーザー数の表示

## 技術スタック

| 役割 | ライブラリ |
|------|-----------|
| Web フレームワーク | Gin |
| WebSocket | gorilla/websocket |
| ORM | GORM |
| DB | PostgreSQL |
| 認証 | bcrypt + Cookie セッション |

## ルーティング

| パス | 説明 |
|------|------|
| `/register` | ユーザー登録 |
| `/login` | ログイン |
| `/logout` | ログアウト |
| `/chat` | チャット画面（要ログイン） |
| `/ws` | WebSocket エンドポイント（要ログイン） |
